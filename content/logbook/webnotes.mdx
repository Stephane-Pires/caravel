### Frontend Security :

- XSS : Inject some malicous script, that will get executed on target clients. [Owasp XSS](https://owasp.org/www-community/attacks/xss/)

- CSRF : Get the credentials of site to fake a user [CSRF](https://en.wikipedia.org/wiki/Cross-site_request_forgery)

- UI Redressing : Make an invisible button when the user click somewhere he clicks on the invisible button [Clickjacking](https://en.wikipedia.org/wiki/Clickjacking)

- (M)an (I)n (T)he (M)iddle : Someone intercept the communication between two endpoint [Man in the middle](https://fr.wikipedia.org/wiki/Attaque_de_l%27homme_du_milieu)

### Image Format :

- PNG : Transparency, heavy format, lossless compression

- JPEG : Lossy compression, support progressive rendering

- WebP : Transparency + progressive rendering, better algo (Lossy & Lossless modes)

- Avif : HDR + Transparency
